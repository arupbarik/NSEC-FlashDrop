# About NSEC FlashDrop
 NSEC FlashDrop is more than just a digital classifieds site; it is a dedicated effort to foster a circular economy within the NSEC campus. By encouraging the reuse of textbooks, electronics, and lab equipment, we reduce environmental waste and lower the financial barrier for students entering new semesters. Our platform shifts the focus from "buying new" to "buying smart," ensuring that high-value resources remain in circulation rather than gathering dust in hostel rooms. This sustainable approach not only preserves the environment but also strengthens the NSEC community by facilitating direct, peer-to-peer support and resource sharing.


## Statement

**The Problem:** The Campus Waste Loop  
Every semester-end at NSEC, high-value assets like textbooks, lab equipment, and hostel essentials are either abandoned or sold to scrap dealers for pennies.

* The Gap: Seniors want to declutter fast; Juniors need affordable gear.

* The Friction: Trust issues, messy coordination, and no central "hub" lead to massive environmental and economic waste.

* The Result: Thousands of usable items end up in landfills while new students overpay for brand-new replacements.

 **The Solution:** NSEC FlashDrop  
  FlashDrop is a high-velocity marketplace built to power the NSEC Circular Economy. By shifting the focus from "selling" to "dropping," we ensure resources stay within the campus gates.

  * ⚡ Urgency-First: Countdown timers push buyers to act fast, solving the "moving out" deadline problem for seniors.
  * 📲 WhatsApp Integrated: One-tap connectivity—no middleman, no delays, just instant peer-to-peer deals.
  * 🌱 Mission LiFE Aligned: Directly reduces the campus carbon footprint by facilitating the reuse of engineering resources.
  * 🏫 Hyper-Local: Designed specifically for NSEC landmarks (Canteen, Common Room, Gate 1) to ensure safe and verified physical handovers.

## ✨ Key Features
* **NSEC centric design:** Optimized categories for engineering essentials—Drawing Boards, Lab Coats, Gate Prep Books, and Scientific Calculators.
* **Seamless WhatsApp Integration:** <br>
* One-Tap Negotiation: No "Add to Cart" or complex checkout. A single click opens a pre-filled WhatsApp chat with the seller.
* Direct P2P: Facilitates instant coordination for price negotiation and meeting spots within the NSEC campus.
* **Mission LiFE Badges**: Users earn "Green Credits" for buying/selling second-hand, aligning with India's sustainability goals.
* **Responsive UI:** A lightweight, fast-loading interface designed for students to browse and "Drop" items while on the move between classes or in the hostel.top.

## 🚀 Getting Started

### Prerequisites
* Node.js (v18+)
* A Supabase Project

### Installation
1.  **Clone the repository:**
    ```bash
    git clone (https://github.com/JustJoyful/NSEC-FlashDrop)
    cd NSEC-FlashDrop
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Setup:**
    Create a `.env.local` file and add your keys:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
    ```
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
